# ⚽ Football Stats Dashboard

A modern football statistics web application built with React and live API integration.

---

## 🔗 Live Demo
https://football-stats-app-snowy.vercel.app/

---

## ✨ Features

- Live football data from API
- League standings table
- Recent matches viewer
- Favorite teams (localStorage)
- Search leagues
- Responsive modern UI

---

## 🛠 Tech Stack

- React (Vite)
- React Router
- JavaScript (ES6+)
- REST API
- Tailwind CSS

---

## 🎯 Purpose

This project demonstrates real-world frontend development skills including API integration, routing, and dynamic UI rendering.
